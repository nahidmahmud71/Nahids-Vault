## 🚀 Overview

Tired of losing crucial algorithms and code snippets in random text files and notes apps, I engineered my own centralized solution. **Nahid's Private Vault** is a personalized cloud storage engine designed to categorize, store, and instantly retrieve code. 

Built completely from scratch without any frontend frameworks, the platform features an advanced glassmorphism UI, a cinematic boot-up sequence, and real-time database integration.

🌐 **Live Website:** [Click Here to Access the Vault](https://nahidmahmud71.github.io/Nahids-Vault/)

---

## ✨ Core Features

* **🎬 Cinematic Intro:** A smooth, immersive loading sequence that sets the sci-fi operating system vibe.
* **🗂️ Language-Specific Domains:** Dedicated, interactive environments for various tech stacks including Java, C++, Python, Web Dev, and SQL.
* **☁️ "Nahid's Cloud" Integration:** A fully functional input terminal to upload, categorize, and archive algorithms and university lab work directly to a secure backend.
* **📡 Code Radar:** A highly responsive, real-time search engine to retrieve specific scripts instantly.
* **📋 One-Click Copy:** Retrieve your code instantly to your clipboard with smart visual feedback.
* **🎨 Cyberpunk UI/UX:** Advanced glassmorphism design, glowing dynamic hover states, and smooth DOM manipulation.

---

## 🛠️ Tech Stack

* **Frontend:** Pure Vanilla HTML5, Custom CSS3 (Animations & Glassmorphism), Vanilla JavaScript (ES6+).
* **Backend & Database:** Google Firebase (Firestore) for real-time cloud data syncing.
* **Icons & Fonts:** FontAwesome 6, Google Fonts (Fira Code, Inter, Montserrat).
